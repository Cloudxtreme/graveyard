:: ChaosBot Required Folders

When you installed your ChaosBot, a list of folders should have also been installed. The list is as follows:

  • commands
  • config
  • data
  • lib
  • logs
  • handlers

These folders are REQUIRED for the bot to run properly. Details on the folders...

commands
This folder is where bot commands are kept.

config
Configuration information.

data
All the bot's data is kept here (including admin/mod lists, warners list, and replies)

lib
The Perl library of modules.

logs
This is where your bot will record conversation logs.

handlers
Handles server information sent by messenger services.

These folders should have all been installed when you installed ChaosBot. If you downloaded this bot in a zip, please check the Chaos AI Technology website for the official installer.
