:: Setup Authorities

      Before ever starting your bot up, it is HIGHLY recommended that you set up some authority users (specifically, the botmasters). Botmasters have access to very powerful commands that could harm your computer if they fall into the wrong hands. You must edit the botmasters list FIRST before signing your bots on!

:: Quick Overview: Client Format

On Leviathan, "clients" (names the bot uses to identify its users), goes in the format of SENDER-username. "SENDER" would be the messenger or medium (the "sender") that the user is talking to the bot on. It should be all uppercase, no punctuation. The "username" is their actual username.

Examples of clients:
MSN Users:
MSN-name@domain.com
MSN-me@passport.net
AIM Users:
AIM-myscreenname
AIM-anothersn
Jabber Users:
JABBER-myname@jabber.com
JABBER-myname@jabber.org

You can get the idea now. When you edit your authority lists, the usernames must be formatted like that. Now, you can continue.

:: Botmasters List

File Path: Leviathan/data/authority/master.txt
Description: You. The bot's owner. This position should be used for nobody else.

This is probably the first file you'll want to edit (and are highly encouraged too).

Following the client format (above), enter YOUR OWN usernames into this list. The Master power is very dangerous in the wrong hands!

:: Administrators List

File Path: Leviathan/data/authority/administrators.txt
Description: Your bot's keepers. Administrators control a lot of your bot, but their powers are limited. They also have access to some commands that can potentially be very annoying to your bot's users. That's why only trusted users should be Admins.

You don't need to edit the admins list at this time, although it wouldn't hurt.

:: Moderators List

File Path: Leviathan/data/authority/moderators.txt
Description: Moderators have very basic bot upkeeping commands. For example, blocking and unblocking users. This is about the extent of the moderator's power. You don't need to edit the mods list at this time, although it wouldn't hurt.
