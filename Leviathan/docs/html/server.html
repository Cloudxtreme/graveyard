:: Server Tools

      In addition to using the HTTP server to chat with your bot, there are other tools provided by the server. Below are listed their request URIs and a description of what they do.

:: Request URIs

/docs
Accesses this documentation from your server. Simply "/docs" will redirect you to index.html, but any subfiles will be shown respectively as well.

/logs
If you enabled HTTP log viewing in your configuration, "/logs" will redirect you to "/logs/chat/convo.html" but other subfiles can be accessed as well (i.e. "/logs/http_access.txt" or "/logs/chat/ChaosAIM/AIM-kirsle.html" for example).

/stats.txt
The server will return the bot's statistics in an easy plain text format (i.e. "BotCount = 5", etc). See below for example output.

/blocked.cgi?user=username
Check the blocked status of a user. username would be their Leviathan name (i.e. "AIM-kirsle" as opposed to just "kirsle"). The server will print "not blocked" if they're clean, or print "1 reason" where 1 is $blocked as returned from &isBlocked and reason is the reason (i.e. "Permanent Ban (BlackList): ...")

/warner.cgi?user=username
Same as blocked.cgi but will return the Warner status of the username. The server will write a 1 (true) or 0 (false)

:: Example stats.txt Output

Version = 1.4
Online = true
Time = 1:20:6:11
SignedOn = 1117497855
BotCount = 4
BotNames = cody@aichaos.com,azulianbot,chaos@aichaos.com,codycks
CommandCount = 103
ClientCount = 1185
LastClient = AIM-kirsle
LastMsgOn = 1117656615
WarnerCount = 2
WarnerPercent = 0.168%
BlacklistCount = 111
BlacklistPercent = 9.367%
BlockedCount = 1
BlockedPercent = 0.084%

Version - The version of the bot.
Online - Should always be "true" - if it wasn't, /stats.txt wouldn't be available would it? :p
Time - Bot's online time (days:hours:minutes:seconds)
SignedOn - The epoch time() of signon.
BotCount - A count of the number of bots signed on.
BotNames - ID's (usernames) of all bots, separated by commas.
CommandCount - The number of (loaded) commands.
ClientCount - Number of your bot's clients (users)
LastClient - The username of the last client to send a message (or blank for none yet).
LastMsgOn - The time() of the last message received (or blank for N/A)
WarnerCount - Number of warners.
WarnerPercent - Percentage of warners to total users.
BlacklistCount - Number of permanently banned users.
BlacklistPercent - Percentage of permanently banned users.
BlockedCount - Number of temporarily blocked users.
BlockedPercent - Percentage of temporarily blocked users.
