:: Setup Bots

      All your bots information are stored in the "bots" folder. Leviathan comes with a sample file in that folder. There is a copy of the file in the "templates" folder.

:: Register Usernames for your Bots!
      In order for your bots to actually sign on to their Instant Messengers, they'll need valid usernames and passwords. Click these links to register usernames for the messengers:
¤ MSN: Register an MSN Passport
¤ AIM: Register an AIM ScreenName
¤ Yahoo: Register a Yahoo ID
¤ Jabber: Download Jabber Messenger to register a username
¤ IRC: Register with /nickserv on your IRC server

      Each bot has its own file inside the "bots" folder. The basic information about the bots (i.e. their brain and replies) are set up first, followed by all the mirrors. Mirrors are basicly just their connections to Instant Messengers. Bots can have one or many mirrors.

      Each mirror contains a list of parameters. These set up the specific parameters of the mirror. Click one of the links below to see a list of the parameters for that connection:

:: Sample AIM Bot
screenname - The bot's screenname that it signs in to AIM with.
password - The bot's AIM password.
autoreconnect - Whether or not the bot will automatically reconnect (1 or 0).
format - The screenname's format (capitilization and spacing).
profile - Path to your bot's AIM profile (HTML document).
icon - Path to your bot's buddy icon (48x48 pixels, <4 KB).
buddies - Path to your bot's buddy list.
chats - Path to a file defining the bot's chat rooms.
chattriggers - Path to a file defining chat triggers.
font - Path to a file defining the bot's font settings.

:: Sample MSN Bot
screenname - Your bot's MSN passport that it uses to sign in to MSN.
password - The bot's MSN password.
nick - The bot's display name.
welcomemsg - Path to bot's welcome message. It sends this when a user starts chatting with the bot. Leave blank for none.
displaypic - Path to bot's display picture.
emoticons - Path to a file defining the bot's custom emoticons.
font - Path to a file defining the bot's font.

:: Sample IRC Bot
screenname - The bot's IRC nickname.
password - (optional) If the username requires a password, enter it there.
server - The IRC server.
port - The IRC server's port. Default is 6667.
channel - Channel to join after the bot connects.

:: Sample Yahoo Bot
screenname - The bot's Yahoo! username.
password - Bot's Yahoo password.
font - Path to a file defining the bot's font.

:: Sample Jabber Bot
screenname - The bot's full Jabber username (i.e. name@jabber.com)
password - The bot's password.
server - The Jabber server (i.e. jabber.com)

:: Sample HTTP Bot
screenname - Host and port of your server (i.e. localhost:2001). Omit port to assume 80.
nick - Nickname to be used in the HTTP chat.

:: Sample SMTP Bot
screenname - Host and port of your server (i.e. localhost:25). Omit port to assume 25.
autoreply - Path to an automated response file. The content of this file is returned to all senders.
addresses - List of addresses to reply to (separate by spaces).

:: Sample TOC Bot
NOTE: The TOC listener was added as an alternative to those who can't get the AIM listener working. Only use TOC as a last resort if you get Net::OSCAR errors.
screenname - The bot's AIM screenname.
password - Bot's AIM password.
profile - Path to the bot's profile (HTML document).
buddies - Path to the bot's buddylist.
chats - Path to a file defining the bot's chat rooms.
font - Path to a file defining the bot's font settings.
