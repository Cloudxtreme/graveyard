:: Get Perl

      AiChaos Leviathan requires a working version of Perl (Practical Extraction and Reporting Language) to be installed on your computer.

You can download ActivePerl from ActiveState.com. Or, go straight to the download page by clicking here.

¤ Required Perl Version: 5.8.00 or higher.

Tip: For Windows users, download the MSI package, it's a self-installer and makes it a lot easier to install Perl.
