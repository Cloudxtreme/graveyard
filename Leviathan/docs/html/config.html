:: Bot Configuration

      The first thing you'll probably need to do is configure your bot. By "configure," I don't necessarily mean "putting in your bot's log-on information." That part comes later.

:: Main Configuration File

      The main configuration file in AiChaos Leviathan is found in "config/settings.cfg" - edit this file with a text editor such as Notepad to configure it.

      Note: This configuration file is Perl code, so you need at least a little bit of knowledge. Here are some little tips, for if you have no experience with Perl codes:

  • A line beginning with the pound sign # are comments. The config file is full of comments to explain what to do. You shouldn't need to touch, edit, or add comments here.
  • Setting a variable goes in the format $variable = 'value'; - that's exactly how they're formatted here.
  • All command lines (lines that don't begin with a #) must END with a semicolon ;
  • The value for a variable can be in "quotes", OR 'apostrophes'. The default settings all use apostrophes, because they're simpler. If the value you want to insert contains an apostrophe itself, put a backslash \ before it. Example:

    # Default sender of e-mails.
    $chaos->{config}->{email}->{from} = 'Dave\'s Bot <bot@davesite.com>';

  • Variables with number values do not need quotes or apostrophes, nor should they. Putting quotes or apostrophes around a number may cause problems later in the bot.
